# Engineering-Thesis 
Piotr Łuczak 246965

Development of a fiber optic system for spatial localization of a person in a room

This page is dedicated to accumulate all the files needed to understand and recreate the project.

The main parts are
- python application for PC
- STM32CUBEIDE project for STM32F303k8
- Datasheets of components used
- KiCad files


# **Disclaimer**

There is a difference in pin connection of STM32 between KiCad schematics and STM32CubeIDE project. Correct connection is presented and explained in STM Project, more precisely, in GPIO library header file. Connection on the PCB was corrected in hardware after production.
